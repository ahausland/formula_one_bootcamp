# formula_one_bootcamp
Formula One data analysis project updated from my CSS M.S. bootcamp final

[f1_racing_display](/f1_racing_display) examines driver and constructor championships, race wins, points throughout the season, and individual drivers. Functions were created to visualize specific seasons or drivers.

[f1_safety_display](/f1_safety_display) examines race incidents (safety car deployments and fatal accidents).

These notebooks were adjusted to display static images compatible with GitHub, so the interactive components of the plotly figures are not viewable. For code without these adjustments, please see the 'clean_plotly' folder.
